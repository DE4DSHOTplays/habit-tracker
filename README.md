## Deployment

This app is deployed on Streamlit Cloud. To deploy your own:

1. Push this repository to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click "New app"
4. Select your GitHub repository and main file (app.py)
5. Click Deploy!

## Database

The app uses SQLite for local data storage. Data is persisted between sessions and automatic backups are created.
